# Two Types of Graphs 
    * Adjacency Matrix 
        + Memory Space - O(|V|^2)
        +Adding A Vertex - O(|V|^2)
        +Adding A Edge - O(1)
        +Removing A Edge - O(1)
        +Removing A Vertex - O(|V|^2)
    * Adjacency List 
        + Memory Space - O(|V| + |E|)
        +Adding A Vertex - O(1)
        +Adding A Edge - O(1)
        +Removing A Edge - O(|E|)
        Removing A Vertix - O(|V|^2)
