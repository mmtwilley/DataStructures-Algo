##Big O
To Push something to end is O(1)
To Remove(Popping) something to the end is O(N)
To Insert something to the front is O(1)
To Remove the first item is O(1)
To Insert in the middle of the array is O(N)
For removal of the linked list is O(N)
Finding an value is O(N)
Finding a reference is O(N)
