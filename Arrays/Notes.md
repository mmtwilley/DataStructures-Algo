## Static Array 
    - can only hold a fixed number of elements
    - allocated in continuous space in memory
        - lookup O(1)
        - push O(1)
        - insert O(n)
        - delete O(n)


## Dynamic Array
  - Creates a new array with a large space
      - lookup O(1)
      - append O(1)
      - insert O(n)
      - delete O(n)
